    React + Vite

    Esta plantilla proporciona una configuración mínima para que React funcione en Vite con HMR y algunas reglas de ESLint.

    Actualmente, hay dos complementos oficiales disponibles:

    • @vitejs/plugin-react utiliza Oxc
    • @vitejs/plugin-react-swc utiliza SWC

    Compilador de React

    El compilador de React no está habilitado en esta plantilla debido a su impacto en el rendimiento de desarrollo y compilación. Para agregarlo, consulte esta documentación .

    Ampliando la configuración de ESLint

    Si estás desarrollando una aplicación de producción, te recomendamos usar TypeScript con reglas de análisis estático de tipos habilitadas. Consulta la plantilla de TypeScript para obtener información sobre cómo integrar TypeScript typescript-eslinten tu proyecto.
