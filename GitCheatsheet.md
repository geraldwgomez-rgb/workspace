# ¿ Que es el commit en GIT HUB ?
## El commit es el momento en el que guardas la informacion de un archivo, para que este publicado en el repositorio y que tu y tus compñaeros de codigo, puedan ver los cambios que hiciste.

# ¿ Que es push y pull ?
## PUSH : Es el acto de enviar tus cambios desde tu computadora local hacia un servidor remoto. 
## PULL : Es el acto de traer y descargar los cambios que otros han echo en el servidor ( GIT HUB ), hacia tu propio servidor o computador.
